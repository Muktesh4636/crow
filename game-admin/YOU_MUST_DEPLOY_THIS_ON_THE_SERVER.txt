This Kokoroko folder is the Android app. It does NOT publish your public web URL by itself.

To see Cricket, Gundu Ata, and Cock fight as THREE SEPARATE blocks on the admin dashboard:

1. Open your game-admin backend project (svs_exch / whatever serves /game-admin/).
2. Find the template or view that renders the page AFTER login at /game-admin/dashboard/
3. Paste the contents of dashboard-three-cards-snippet.html:
   - Put the entire <style>...</style> block in your base layout or dashboard template head (once).
   - Put the HTML from <!-- START paste --> through <!-- END paste --> inside the dashboard body.
4. Change href on each card to match your real routes if they differ.
5. Deploy that backend to your production host (git pull, restart server, etc.)

Until step 3–5 happen on the SERVER, the live site will not change.
